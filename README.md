# course_platform - System Zarządzania Kursami

Aplikacja wspomaga zarządzanie kursami online, zapisywanie użytkowników oraz wystawianie recenzji.

### Opis projektu
Aplikacja wspomaga zarządzanie kursami online, zapisywanie użytkowników oraz wystawianie recenzji.



**Główne funkcjonalności:**
1. Walidacja wieku (18+) i formatu email przy tworzeniu konta.
2. System zniżek: -20% dla kont Premium i -10 PLN dla użytkowników z regionu PL.
3. Blokada zapisu na nieopublikowane kursy.
4. Blokada ponownego zapisu na ten sam kurs.
5. System recenzji dostępny tylko dla osób, które zapisały się na dany kurs.
6. Integracja z zewnętrznym systemem Email (mockowana w testach).
