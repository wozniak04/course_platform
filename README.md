# course_platform - System Zarządzania Kursami

Aplikacja wspomaga zarządzanie kursami online, zapisywanie użytkowników oraz wystawianie recenzji.

### Autor
- **Mikołaj Woźniak** 
- **Grupa:** 4

### Opis projektu
Aplikacja do zarządzanie kursami online, zapisywanie użytkowników na kurs,tworzenie kursu oraz wystawianie recenzji.

### Instrukcja uruchomienia

1. **Instalacja zależności:**
   pip install -r requirements.txt
2. **Odpalanie flaska:**
   na linux python3 -m flask --app app/api.py --debug run.
   na windows: python -m flask --app app/api.py --debug run.
3. **Odpalanie testów:**
   na linux python3 -m pytest. 
   na windows: python -m pytest.
